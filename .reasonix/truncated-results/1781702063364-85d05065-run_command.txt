📁 aiops - AIOps 智能运维相关操作
  ├── 📂 alarm
  │       ├─── ack - 确认告警
  │       ├─── clear - 清除告警
  │       ├─── list - 查询告警信息(当前告警,可选择是否包含历史告警)
  │       ├─── unack - 取消确认告警
  ├── 📂 check_policy
  │       ├─── delete - 删除检查策略
  │       ├─── disable - 禁用检查策略
  │       ├─── enable - 启用检查策略
  │       ├─── execute - 执行检查策略
  │       ├─── list - 查询检查策略列表
  ├── 📂 check_result
  │       ├─── list - 查询检查策略异常检查结果列表
  │       ├─── show - 查询检查策略异常检查结果详情
  ├── 📂 diagnose_task
  │       ├─── create - 创建智能分析任务
  │       ├─── status - 查询性能诊断任务状态
  ├── 📂 health
  │       ├─── query_data - 查询健康度相关数据（容量预测/性能预测/性能异常）
  │       ├─── show_detail - 查询健康维度的扣分详情
  │       ├─── show_score - 查询对象健康度
  ├── 📂 performance
  │       ├─── create_collect_task - 创建性能文件收集任务
  │       ├─── download_collect_result - 下载性能文件
  │       ├─── list_indicators - 列出监控对象类型支持的监控指标
  │       ├─── list_object_types - 获取所有监控对象类型
  │       ├─── query - 查询历史性能数据
  │       ├─── show_indicators - 显示监控指标详细信息
  ├── 📂 topology
  │       ├─── query_graph_path - 查询拓扑图库信息（支持 NAS、K8s、DB 等业务类型）
  │       ├─── query_luns - 查询拓扑图 LUN 列表
  │       ├─── query_san_path - 查询 SAN 路径拓扑结构（支持 IP_SAN 和 FC_SAN）
  │       ├─── query_vms - 查询拓扑图虚拟机和虚拟磁盘列表，或查询 BMS 下物理磁盘列表
