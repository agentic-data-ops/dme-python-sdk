
============================================================
动作：protect snapshot_group create
============================================================

描述:
  创建快照一致性组

详细说明:
  创建快照一致性组

参数说明:
------------------------------------------------------------

  --name
      快照一致性组名称

  --protect_group_id
      保护组的 ID

  --description
      描述信息

  --creation_mode
      创建模式，可选值：new_snapshot

输出说明:
------------------------------------------------------------
    {
        task_id: 任务ID (string, 1~64个字符),
    }

============================================================
使用示例:
  pydme protect snapshot_group create
  pydme protect snapshot_group create --name <value> --protect_group_id <value> --description <value> --creation_mode <value>
============================================================
