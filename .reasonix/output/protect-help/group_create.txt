
============================================================
动作：protect group create
============================================================

描述:
  创建保护组

详细说明:
  创建保护组

参数说明:
------------------------------------------------------------

  --name
      保护组名称

  --storage_id
      存储设备 ID

  --lun_ids
      LUN 的 ID 列表，条件必选，当基于 LUN 创建保护组时为必传字段

  --lun_group_id
      LUN 组 ID，条件必选，当基于 LUN 组形式创建保护组时为必传字段

  --description
      保护组描述

输出说明:
------------------------------------------------------------
    {
        id: 保护组ID (string),
        task_id: 任务ID (string, 1~64个字符),
    }

============================================================
使用示例:
  pydme protect group create
  pydme protect group create --name <value> --storage_id <value> --lun_ids <value> --lun_group_id <value> --description <value>
============================================================
