
============================================================
动作：system tag list
============================================================

描述:
  批量查询标签

详细说明:
  批量查询标签

参数说明:
------------------------------------------------------------

  --start
      分页起始位置，默认 1

  --limit
      分页数量，默认 100

  --name
      标签名称过滤（可选）

  --tag_type_id
      标签类型 ID 过滤（可选）

输出说明:
------------------------------------------------------------
    {
        total: 标签数量 (int32),
        datas: 标签列表 (List<Tag>)。参数格式如下：[{
            id: 标签ID (string),
            name: 标签名称 (string),
            tag_type_name: 标签类型名称 (string),
        }, ...],
    }

============================================================
使用示例:
  pydme system tag list
  pydme system tag list --start <value> --limit <value> --name <value> --tag_type_id <value>
============================================================
