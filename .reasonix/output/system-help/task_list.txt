
============================================================
动作：system task list
============================================================

描述:
  批量查询任务

详细说明:
  批量查询任务

参数说明:
------------------------------------------------------------

  --start
      分页起始位置，默认 1

  --limit
      分页数量，默认 100

  --task_name
      任务名称过滤（可选）

  --status
      状态过滤（可选，1-初始状态;2-执行中;3-成功;4-部分成功;5-失败;6-超时）

  --owner_id
      创建任务用户 ID 过滤（可选）

  --create_time_from
      创建时间起始（可选，UTC 毫秒数）

  --create_time_to
      创建时间结束（可选，UTC 毫秒数）

输出说明:
------------------------------------------------------------
    {
        total: 任务数量 (int32),
        tasks: 任务列表 (List<TaskDetail>)。参数格式如下：[{
            id: 任务ID (string),
            name_en: 任务英文名称 (string),
            status: 状态 (string),
            progress: 任务进度 (string),
            owner_name: 创建任务用户名称 (string),
            create_time: 创建时间 (string),
            start_time: 开始时间 (string),
            end_time: 结束时间 (string),
        }, ...],
    }

============================================================
使用示例:
  pydme system task list
  pydme system task list --start <value> --limit <value> --task_name <value> --status <value> --owner_id <value> --create_time_from <value> --create_time_to <value>
============================================================
