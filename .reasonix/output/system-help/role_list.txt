
============================================================
动作：system role list
============================================================

描述:
  批量查询角色信息

详细说明:
  批量查询角色信息。

参数说明:
------------------------------------------------------------

  --page_no
      页数 (必选, integer, 最小值：1)。默认值：1

  --page_size
      页面大小 (必选, integer, 5~100)。默认值：10

  --name
      角色名搜索关键字 (可选, string, 最多64个字符)

输出说明:
------------------------------------------------------------
    {
        total: 总数 (integer, 最大值：10),
        datas: 角色数据 (List<RoleData>, 数组最大成员个数：5000)。参数格式如下：[{
            id: 角色ID (integer, 1~2147483647),
            name: 角色名称 (string, 最多64个字符),
            description: 描述 (string, 最多127个字符),
        }, ...]
    }

============================================================
使用示例:
  pydme system role list
  pydme system role list --page_no <value> --page_size <value> --name <value>
============================================================
