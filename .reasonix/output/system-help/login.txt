
============================================================
动作：system login
============================================================

描述:
  认证用户登录

详细说明:
  认证用户登录
  强制调用 client.login() 完成认证，然后从 header 获取 accessSession，
  提示用户可配置环境变量复用认证密钥，避免重复登录。

参数说明:
------------------------------------------------------------

输出说明:
------------------------------------------------------------
    {
        task_id: 任务ID (string, 1~64个字符),
    }，包含 accessSession
    - accessSession: 会话 token，用于后续请求的 X-Auth-Token header

============================================================
使用示例:
  pydme system login
  pydme system login 
============================================================
